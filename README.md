# RA — Raciocínio Algorítmico · PUCPR

Ferramenta de estudo interativa para a disciplina de **Raciocínio Algorítmico** da PUCPR.

## Como usar

Abra `index.html` direto no navegador — sem servidor, sem dependências locais.

## Features

- **Python no browser** via Skulpt — execute código real sem instalar nada
- **60+ exercícios** cobrindo todos os tópicos do GitHub do professor
- **3 Simulados oficiais** com Testes de Mesa (TRACE), Debug e Implementação
- **Modo Drill** — pratique linha a linha até decorar os padrões
- **Cheat sheets** por tópico — referência rápida ao clicar no tópico
- **Filtro 🔄** — mostre só o que ainda precisa revisar
- **🎲 Surpreenda-me** — exercício aleatório com peso maior nos marcados para rever
- **Contagem regressiva** até a prova
- Progresso salvo no localStorage
